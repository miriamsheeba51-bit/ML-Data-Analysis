# Indian Traffic Violations Data Analysis

## Objective
Analyze Indian traffic violation data using Python to identify violation patterns, driver behavior, and traffic safety trends.

## Dataset
Indian_Traffic_Violations.csv

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Duplicate Removal
- Statistical Analysis
- Traffic Violation Analysis
- Fine Amount Analysis
- Driver Demographic Analysis
- Vehicle Analysis
- Data Visualization

## Key Insights
- Most common traffic violations
- Fine amount statistics
- Speeding analysis
- Helmet and seatbelt compliance
- Driver age and gender distribution
- State-wise traffic violations
- Vehicle type distribution
